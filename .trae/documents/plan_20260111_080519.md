# PyQt5 替换为 PySide6 修改计划

## 修改范围

### 1. 修改 requirements.txt
- 将 `PyQt5>=5.15.6` 替换为 `PySide6>=6.5.0`

### 2. 修改 sender/sender_gui.py
- 将所有 `PyQt5` 导入替换为 `PySide6`
- 将 `pyqtSignal` 替换为 `Signal`
- 具体修改：
  - 第3-8行：导入语句替换
  - 第9行：`from PyQt5.QtCore import Qt, QThread, pyqtSignal, QTimer` → `from PySide6.QtCore import Qt, QThread, Signal, QTimer`
  - 第19行：`status_update = pyqtSignal(dict)` → `status_update = Signal(dict)`
  - 第20行：`finished = pyqtSignal()` → `finished = Signal()`

### 3. 修改 receiver/receiver_gui.py
- 将所有 `PyQt5` 导入替换为 `PySide6`
- 将 `pyqtSignal` 替换为 `Signal`
- 具体修改：
  - 第3-8行：导入语句替换
  - 第9行：`from PyQt5.QtCore import Qt, QThread, pyqtSignal, QTimer` → `from PySide6.QtCore import Qt, QThread, Signal, QTimer`
  - 第11行：`from PyQt5 import QtWidgets` → `from PySide6 import QtWidgets`
  - 第29行：`packet_received = pyqtSignal(object)` → `packet_received = Signal(object)`

## 修改步骤

1. 首先修改 requirements.txt 文件
2. 然后修改 sender_gui.py 文件
3. 最后修改 receiver_gui.py 文件

## 预期效果
- 所有导入语句替换完成
- 信号定义从 `pyqtSignal` 替换为 `Signal`
- 代码能够正常运行，功能保持不变
- 支持 Windows ARM64 架构
